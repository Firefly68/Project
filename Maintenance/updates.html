Windows 10 no longer supports setting updates to download but not install, or the option to have them only delivered when you manually search for them (at least not easily, there are hacks but they are not recommended for non-technical computer users). You can however still adjust some settings, such as usage times, whether you get just Windows updates ro all Microsoft product updates, or force an out of schedule check to see if there are any missing patches.

Note that Microsft also currently releases Windows patches as a bundle rather than multiple files. As of the time of writing this, there is no easy way to install a single patch for a specific problem without going through various Microsoft troubleshooting channels and their support site. Some urgent patches can be released on a non-standard schedule but the majority of non-critical issues are all released in the monthly patch.
